(static) createWorkspace(parameters, {@ui5/fs/AbstractReader}, {@ui5/fs/AbstractReaderWriter})

Source:
  • ui5-fs/lib/resourceFactory.js, line 149
Creates a Workspace A workspace is a DuplexCollection which reads from the project sources. It is used during the build process to write modified files into a separate writer, this is usually a Memory adapter. If a file already exists it is fetched from the memory to work on it in further build steps.
Parameters:
Name Type Attributes Default Description
parameters object
{@ui5/fs/AbstractReader} parameters.reader Single reader or collection of readers
{@ui5/fs/AbstractReaderWriter} [parameters.writer] A ReaderWriter instance which is only used for writing files. If not supplied, a Memory adapter will be created.
parameters.name string <optional>
 "vir & fs source" Name of the collection
parameters.virBasePath string <optional>
 "/" Virtual base path
Returns:
{@ui5/fs/DuplexCollection} DuplexCollection which wraps the provided resource locators
